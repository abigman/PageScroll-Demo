# PageScroll-Demo


PageScroll-Demo like Tencent news.

## 简介

仿照腾讯新闻客户端的一个课程表 demo，下面有预览图，不过实际项目中的实现方法不太一样。

大体是一样的啦~大家做个参考就好。

## 运行环境

* Xcode 5 及以上
* iOS 7 及以上

项目设置的最低要求是 iOS8，实际上改成 7.0 也能运行，没有用新的 API。
